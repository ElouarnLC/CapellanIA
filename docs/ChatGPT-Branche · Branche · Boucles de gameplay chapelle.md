# 1. La boucle principale que je vous recommande

## Fonder → doter → desservir → embellir → consacrer → transmettre

Une chapelle ne devrait pas être un bâtiment procurant immédiatement des bonus. Elle deviendrait progressivement une **institution fonctionnelle**.

### 1. Fonder

Le joueur choisit :

- son emplacement ;
- son statut : oratoire privé, chapelle castrale, chapelle palatine, chapelle funéraire ;
- sa dédicace ;
- son rapport à la paroisse locale ;
- sa fonction dominante.

Exemples de fonctions :

- **chapelle domestique** : piété personnelle, diminution du stress, événements familiaux ;
- **chapelle dynastique** : légitimité, mémoire des ancêtres, sépultures ;
- **chapelle de cour** : prestige, chantres, diplomatie et cérémonial ;
- **chapelle reliquaire** : pèlerinage, miracles, revenus et convoitises ;
- **chapelle d’ordre chevaleresque** : chevalerie, compagnonnage, tournois ;
- **chapelle expiatoire** : réduction de l’accomplissement spirituel négatif ou traitement de certains péchés.

### 2. Obtenir l’autorisation

La fondation ouvre une négociation avec :

- le prêtre paroissial ;
- l’évêque ou l’archevêque ;
- éventuellement un chapitre cathédral ou une abbaye détentrice de droits.

Le joueur peut :

- demander une licence canonique ;
- payer une compensation annuelle ;
- concéder une part des offrandes ;
- limiter les offices autorisés ;
- utiliser un lien, un secret ou un puppet ecclésiastique ;
- construire clandestinement.

Le système annoncé de **puppet chaplain** paraît particulièrement pertinent : un chapelain soumis pourrait faciliter la légalisation d’une fondation, intercéder auprès de l’archidiocèse ou couvrir certains abus. ([note（ノート）](https://note.com/tsukuda_msh/n/n3cc80824a966?hl=en&utm_source=chatgpt.com))

### 3. Doter

La chapelle doit recevoir une dotation durable :

- rente foncière ;
- moulin, vigne ou péage ;
- somme annuelle ;
- objets liturgiques ;
- logement du chapelain ;
- droit de prélèvement particulier.

Une forte dotation facilite le recrutement, mais suscite :

- convoitises ;
- détournements ;
- nominations politiques ;
- contestations successorales ;
- demandes de taxation ecclésiastique.

### 4. Recruter le personnel

Une chapelle n’est active que si elle possède au minimum :

- un chapelain ;
- les ornements nécessaires ;
- un autel utilisable.

Les chapelles plus prestigieuses peuvent engager :

- plusieurs chapelains ;
- chantres ;
- clercs ;
- enfants de chœur ;
- sommeliers ou gardiens des ornements ;
- maître de chapelle ;
- maître des œuvres ;
- artistes attachés temporairement.

### 5. Faire fonctionner la chapelle

Chaque année, le joueur arbitre entre :

- dépenses ordinaires ;
- célébrations ;
- embellissements ;
- entretien ;
- rémunération du personnel ;
- conservation ou transport du trésor.

Une chapelle mal entretenue perd progressivement sa qualité et peut devenir :

- insuffisamment desservie ;
- dépouillée ;
- profanée ;
- contestée ;
- inutilisable.

### 6. Transmettre

À la mort du fondateur, la chapelle devient un enjeu successoral :

- l’héritier respecte-t-il les obligations ?
- les messes anniversaires sont-elles encore célébrées ?
- les objets suivent-ils le souverain ou restent-ils dans l’édifice ?
- les armoiries du prédécesseur sont-elles conservées ou remplacées ?
- une nouvelle branche dynastique ajoute-t-elle ses propres décors ?

Cette dernière phase donne de la profondeur temporelle au système.

# 2. Amélioration des boucles déjà indiquées en violet

## A. « Jouer le chapelain »

L’idée est bonne, mais le chapelain ne devrait pas être simplement une version inférieure de l’évêque. Il lui faut un gameplay propre fondé sur une **double dépendance** :

- dépendance canonique envers l’évêque ;
- dépendance économique et politique envers le seigneur.

### Ressources ou jauges possibles

**Faveur du seigneur**

Elle augmente avec :

- offices privés ;
- discrétion ;
- soutien politique ;
- missions diplomatiques ;
- glorification de la dynastie ;
- services personnels.

**Réputation ecclésiastique**

Elle augmente avec :

- résidence effective ;
- célébration régulière ;
- respect des interdits ;
- érudition ;
- participation aux synodes ;
- relations avec l’évêque.

**Crédit liturgique**

Il représente la qualité quotidienne du service :

- nombre de messes accomplies ;
- ponctualité ;
- qualité des chantres ;
- état des ornements ;
- respect du calendrier.

Le joueur chapelain doit donc continuellement arbitrer entre son patron et l’institution ecclésiale.

### Progression de carrière

Une carrière possible :

**clerc → chantre ou chapelain auxiliaire → chapelain titulaire → premier chapelain → maître de chapelle → évêque ou dignitaire de cour.**

Les personnages les plus capables peuvent être envoyés :

- chercher une relique ;
- négocier avec un évêque ;
- accompagner une ambassade ;
- tenir les comptes du trésor ;
- fonder une nouvelle chapellenie ;
- représenter le prince à un conseil.

Cela correspond bien à vos exemples de chapelains et chantres employés comme ambassadeurs ou hommes de confiance.

---

## B. « Réaliser des sacrements pour gagner de l’argent »

Il faut éviter d’en faire une simple décision répétable. Transformez-la en **économie de juridiction**.

Le chapelain peut empiéter progressivement sur les droits paroissiaux :

1. bénédictions et offices non contestés ;
2. mariages privés ;
3. baptêmes ;
4. confession ou viatique ;
5. funérailles et sépultures ;
6. captation durable des offrandes.

Chaque empiètement apporte :

- argent ;
- influence locale ;
- faveur du seigneur ;
- parfois opinion des courtisans.

Mais il augmente une jauge de **conflit paroissial**.

À certains seuils :

- protestation du curé ;
- arbitrage de l’archidiacre ;
- enquête épiscopale ;
- suspension du chapelain ;
- interdit sur la chapelle ;
- procès canonique ;
- excommunication dans les cas les plus graves.

Le joueur peut désamorcer le conflit en versant une redevance ou en signant un accord délimitant précisément les droits.

Ainsi, la boucle devient :

**empiéter → profiter → être dénoncé → négocier ou résister → obtenir éventuellement un privilège permanent.**

## C. « Nommer sans autorisation épiscopale »

Il faut distinguer plusieurs types de candidats :

| Candidat | Avantage | Risque |
|---|---|---|
| Prêtre reconnu | Faible conflit canonique | Moins dépendant |
| Parent du seigneur | Loyauté dynastique | Népotisme, faible compétence |
| Clerc non ordonné | Facile à contrôler | Scandale et invalidité |
| Prêtre vagabond | Peu coûteux | Absence, corruption |
| Savant étranger | Haute érudition | Coût et méfiance locale |
| Protégé de l’évêque | Soutien ecclésiastique | Faible contrôle seigneurial |

La nomination clandestine devrait apporter un bénéfice immédiat, mais créer un **statut irrégulier** visible dans les événements et les conseils synodaux.

Le souverain pourrait ensuite :

- faire régulariser son candidat ;
- falsifier ou acheter une autorisation ;
- intimider l’évêque ;
- remplacer discrètement le chapelain ;
- défendre son droit de patronage ;
- laisser le conflit dégénérer.

---

## D. « Conseils synodaux »

C’est une excellente idée, mais la chapelle doit y apparaître comme un dossier concret, pas seulement dans un événement textuel.

Un synode pourrait examiner :

- les chapelles non autorisées ;
- le cumul des bénéfices ;
- les chapelains non résidents ;
- les clercs non ordonnés ;
- les messes non célébrées ;
- l’état des objets sacrés ;
- les droits paroissiaux ;
- la rémunération insuffisante ;
- l’usage profane de la chapelle.

Le joueur ecclésiastique choisit une politique :

- tolérance pragmatique ;
- réforme ;
- répression ;
- fiscalisation ;
- contrôle centralisé.

Le seigneur peut :

- défendre son chapelain ;
- produire ses privilèges ;
- corrompre un enquêteur ;
- promettre une nouvelle dotation ;
- présenter le curé comme jaloux ;
- accepter une visite canonique.

Les archidiocèses de *By God Alone* devraient posséder une autorité géographique distincte des frontières séculières, ce qui rend ce type de conflit particulièrement naturel. ## E. « Érudition + vertus »

Ne faites pas reposer le recrutement uniquement sur l’Érudition. Chaque personnel devrait répondre à un profil.

### Chapelain

- Érudition : doctrine, qualité des sermons, résistance aux enquêtes ;
- Diplomatie : apaiser le curé et les courtisans ;
- Intrigue : dissimuler les irrégularités et servir le prince ;
- Foi ou accomplissement spirituel : qualité morale du service ;
- traits vertueux : réputation ;
- traits pécheurs : événements risqués, mais parfois utiles au patron.

### Chantre

- Érudition ;
- traits liés à la voix ou à la santé ;
- culture ;
- prestige ;
- compatibilité linguistique et liturgique.

### Maître des œuvres

- Intendance ;
- compétence d’artisan ;
- connaissance architecturale ;
- capacité à respecter les délais.

Un chapelain très pieux mais indépendant peut devenir une menace politique. Un chapelain corrompu mais docile peut être extrêmement utile.

---

## F. « Être bien vu du seigneur »

Transformez cette idée en compétition interne à la chapelle.

Les chapelains et chantres se disputent :

- les gratifications ;
- les bénéfices ;
- les missions diplomatiques ;
- la garde des reliques ;
- le titre de premier chapelain ;
- l’accès à l’oratoire privé ;
- la proximité physique avec le souverain.

Un membre peut :

- dénoncer l’incompétence d’un rival ;
- embellir exceptionnellement une cérémonie ;
- composer un office pour un événement dynastique ;
- cacher un vol ;
- obtenir le soutien de la reine ;
- s’attribuer le mérite d’une relique nouvellement acquise.

Le titre de **maître de chapelle** peut devenir une distinction de cour, avec possibilité d’être retiré arbitrairement.

---

## G. « Récompenses de Noël »

Plutôt qu’un événement isolé, utilisez une cérémonie annuelle de rétribution.

À Noël, Pâques ou lors de la fête patronale, le seigneur répartit :

- argent ;
- étoffes ;
- robes ;
- bénéfices ;
- logement ;
- terres ;
- objets personnels ;
- promotions.

Une récompense généreuse :

- fidélise le personnel ;
- augmente le prestige de la chapelle ;
- attire de meilleurs chantres.

Une récompense insuffisante provoque :

- départs ;
- absentéisme ;
- vols ;
- mauvaise qualité du service ;
- chants satiriques ou indiscrétions ;
- recrutement par une cour concurrente.

---

## H. « Mettre les armoiries »

Il faut dépasser le simple choix cosmétique. Le décor héraldique doit devenir un acte politique.

Le joueur choisit où placer :

- ses armes ;
- celles de son conjoint ;
- celles de sa mère ;
- celles d’un lignage revendiqué ;
- celles du fondateur ;
- celles d’un ordre chevaleresque ;
- celles d’un allié ou bienfaiteur.

Les supports possibles :

- vitraux ;
- clefs de voûte ;
- parements ;
- chasubles ;
- courtines ;
- pavements ;
- retables ;
- clôtures ;
- livres.

### Effets possibles

- légitimité dynastique ;
- opinion du conjoint ou de sa famille ;
- consolidation d’une revendication ;
- mémorialisation d’une alliance ;
- rivalité entre branches ;
- scandale en cas d’armes usurpées ;
- damnatio memoriae après une rébellion.

Un nouveau seigneur pourrait :

- conserver l’ancien décor ;
- le compléter ;
- le recouvrir ;
- le briser ;
- replacer ses armes au-dessus de celles de son prédécesseur.

C’est une boucle très « CK3 » parce qu’elle matérialise les relations dynastiques.

## I. « Ancrage géographique de la commande »

Cette idée doit devenir une **chaîne de production artistique**.

Chaque région peut disposer de traditions :

- sculpture sur pierre ;
- peinture ;
- vitrail ;
- broderie ;
- orfèvrerie ;
- menuiserie ;
- chant liturgique.

Le joueur peut :

- commander localement : moins cher, plus rapide, style régional ;
- recruter un artiste de cour : plus cohérent, entretien coûteux ;
- faire venir un maître étranger : prestige élevé, retard et risques de voyage ;
- acheter une œuvre existante ;
- piller ou récupérer une œuvre ;
- recevoir un don diplomatique.

Le résultat dépend de :

- la qualité de l’artiste ;
- la distance ;
- le matériau disponible ;
- la sécurité des routes ;
- la précision de la commande ;
- le contrôle du maître des œuvres.

Événements :

- œuvre endommagée pendant le transport ;
- artiste retenu par un autre prince ;
- surcoût ;
- matériau remplacé ;
- iconographie jugée inconvenante ;
- œuvre exceptionnellement réussie ;
- diffusion du style dans la région.

## J. « Décor suivant le calendrier liturgique »

Le changement de parements ne doit pas devenir du micromanagement quotidien.

Je recommande des **ensembles liturgiques**, équipables automatiquement selon les saisons :

- ensemble ordinaire ;
- ensemble festif ;
- ensemble pénitentiel ;
- ensemble funéraire ;
- ensemble patronal ;
- ensemble dynastique ou chevaleresque.

Chaque ensemble comprend abstraitement :

- parement d’autel ;
- chasuble ;
- courtines ;
- textiles ;
- éventuellement objets ou livres.

Le joueur prépare un ensemble, puis le personnel l’emploie automatiquement.

Lors des grandes fêtes, plusieurs choix apparaissent :

- utiliser l’ensemble approprié ;
- sortir un trésor exceptionnel ;
- improviser avec un ensemble inadapté ;
- emprunter à une abbaye ;
- laisser la cérémonie se tenir pauvrement.

Le bonus dépend de la correspondance entre :

- occasion ;
- couleur ou iconographie ;
- qualité ;
- état de conservation ;
- compétence du personnel.

## K. « Chapelains dans la suite de voyage »

C’est l’une des meilleures boucles du document.

Lors d’un voyage, le joueur compose une **chapelle itinérante** :

- chapelain ;
- chantres ;
- clerc du trésor ;
- reliquaire ;
- vêtements ;
- autel portatif ;
- livres ;
- objets précieux.

### Avantages

- offices pendant le voyage ;
- réduction du stress ;
- accomplissement spirituel ;
- funérailles ou sacrements d’urgence ;
- prestige lors des étapes ;
- possibilité de consacrer ou bénir ;
- amélioration de certaines activités.

### Coûts et risques

- encombrement ;
- vitesse réduite ;
- salaire ;
- vol ;
- perte ou bris d’une relique ;
- attaque du convoi ;
- chapelain malade ;
- conflit avec le clergé local ;
- nécessité d’obtenir un lieu de célébration.

Le joueur peut voyager léger ou transporter tout son trésor. Cela crée une véritable décision stratégique.

## L. « Chapelle dans le tournoi »

La chapelle peut intervenir à plusieurs moments :

### Avant

- veillée spirituelle ;
- bénédiction des armes ;
- messe de l’ordre ;
- exposition des reliques ;
- serment chevaleresque.

### Pendant

- prière avant une épreuve ;
- refuge ;
- contestation d’un serment ;
- chapelain assistant un blessé.

### Après

- action de grâce ;
- funérailles ;
- fondation d’une messe pour un mort ;
- offrande du vainqueur ;
- exposition d’un trophée.

Une chapelle construite spécialement pour l’événement pourrait procurer un prestige considérable, avec le risque qu’elle soit inachevée, mal consacrée ou trop coûteuse.

# 3. Nouvelles boucles de gameplay

## 1. La dédicace et le saint patron

Au moment de fonder la chapelle, le joueur choisit un titulaire :

- Vierge ;
- saint dynastique ;
- saint local ;
- saint guerrier ;
- saint guérisseur ;
- saint royal ;
- saint lié à une relique.

La dédicace influence :

- les événements ;
- les pèlerins ;
- le type de relique recherché ;
- certaines activités ;
- l’opinion des populations locales ;
- la fête patronale annuelle.

Le prestige de la chapelle augmente lorsque son décor, ses reliques et ses offices forment un programme cohérent.

Exemple :

**Saint Michel + ordre chevaleresque + armes dynastiques + relique militaire + messe avant les tournois.**

Cette cohérence pourrait former une statistique de **programme dévotionnel**.

---

## 2. La fabrique de la mémoire dynastique

Lorsqu’un membre de la dynastie meurt, le joueur décide :

- de créer une messe anniversaire ;
- d’ajouter son portrait ;
- de commander un vitrail ;
- de placer son cœur ou ses entrailles dans la chapelle ;
- de déposer un objet lui ayant appartenu ;
- d’inscrire son nom au nécrologe ;
- de supprimer la mémoire d’un traître.

Plus une mémoire est entretenue, plus elle fournit :

- légitimité ;
- splendeur dynastique ;
- réduction du stress lors des deuils ;
- opinion des parents ;
- propagation d’une légende.

Mais chaque fondation augmente les charges annuelles. Le joueur peut se retrouver avec trop de messes promises et pas assez de personnel.

La boucle devient :

**promettre des offices → acquérir de la légitimité → accumuler des obligations → réformer ou trahir les fondations.**

---

## 3. La surcharge liturgique

Chaque chapelle possède une **capacité de service**, calculée selon :

- nombre de chapelains ;
- livres ;
- autels ;
- personnel ;
- dotation ;
- compétence.

Les obligations consomment cette capacité :

- offices quotidiens ;
- messes anniversaires ;
- services du souverain ;
- fêtes ;
- funérailles ;
- fondations privées.

Si la capacité est dépassée :

- offices bâclés ;
- absentéisme ;
- messes fictivement déclarées ;
- mécontentement des fondateurs ;
- enquête ;
- perte d’accomplissement spirituel.

Cela crée une raison mécanique d’engager davantage de personnel.

---

## 4. La visite canonique

Périodiquement, un visiteur ecclésiastique inspecte la chapelle.

Il vérifie :

- autorisation ;
- ordination ;
- résidence ;
- inventaire ;
- revenus ;
- état des reliques ;
- célébration des fondations ;
- comportement du personnel.

Le joueur peut :

- ouvrir les comptes ;
- préparer une mise en scène ;
- dissimuler un objet ;
- corrompre le visiteur ;
- invoquer un privilège ;
- produire un faux ;
- rejeter son autorité.

La qualité de l’administration et la compétence du chapelain déterminent les résultats.

---

## 5. L’inventaire et le trésor

Les objets peuvent se répartir entre :

- trésor fixe de la chapelle ;
- garde-robe itinérante ;
- collection personnelle du seigneur ;
- dépôt auprès d’un monastère ;
- trésor ecclésiastique.

Lors d’une succession, d’une guerre ou d’un voyage, des objets peuvent :

- disparaître ;
- être déplacés ;
- être fondus ;
- être vendus ;
- être revendiqués par l’évêque ;
- suivre un héritier secondaire ;
- être cachés par le gardien.

Le personnage responsable du trésor devient donc important.

---

## 6. Reliques : authenticité, culte et concurrence

Une relique peut avoir plusieurs caractéristiques :

- prestige supposé ;
- authenticité ;
- réputation publique ;
- provenance ;
- adéquation avec la dédicace ;
- état du reliquaire.

Le joueur peut :

- acheter une relique ;
- envoyer un chapelain la négocier ;
- la recevoir comme cadeau ;
- la subtiliser ;
- partager ou fragmenter une relique ;
- faire authentifier un objet douteux ;
- promouvoir un miracle.

Une relique douteuse peut néanmoins devenir puissante si sa réputation est bien entretenue.

La chapelle peut alors évoluer en lieu de pèlerinage, en cohérence avec les systèmes annoncés autour des cathédrales, saints et nouveaux lieux saints. ([GamingOnLinux](https://www.gamingonlinux.com/2026/06/crusader-kings-iii-by-god-alone-expansion-coming-september-30/?utm_source=chatgpt.com))

---

## 7. Le miracle local

Lors d’une maladie, d’un siège ou d’un accident :

- la relique est exposée ;
- une procession est organisée ;
- un vœu est formulé ;
- un miracle est ensuite revendiqué.

Le joueur choisit :

- prudence ;
- promotion active ;
- enquête ;
- instrumentalisation politique.

Conséquences :

- afflux de pèlerins ;
- revenus ;
- réputation du saint ;
- opposition de l’évêque ;
- accusation de fraude ;
- promotion éventuelle de la chapelle.

---

## 8. La rivalité des chapelles

Dans une même cour peuvent coexister :

- chapelle principale ;
- oratoire du souverain ;
- oratoire du conjoint ;
- chapelle basse des officiers ;
- chapelle funéraire d’une branche dynastique.

Les rivalités concernent :

- accès au souverain ;
- priorité cérémonielle ;
- possession des reliques ;
- personnel ;
- financement ;
- lieu de sépulture ;
- représentation héraldique.

Une reine peut développer son propre oratoire et son propre réseau de chapelains, créant un foyer politique parallèle.

---

## 9. Chapelle et factions de cour

Le personnel de la chapelle peut constituer un petit réseau :

- chapelain proche du souverain ;
- chantres issus d’une région étrangère ;
- parents placés dans des bénéfices ;

# 7. Le principe de design à conserver

Chaque amélioration devrait procurer **un avantage visible mais créer simultanément une nouvelle obligation** :

- davantage de reliques → plus de prestige, mais plus de risques de vol ;
- davantage de messes → plus de légitimité, mais plus de personnel nécessaire ;
- chapelain docile → plus de contrôle, mais moins de réputation canonique ;
- artiste étranger → plus de prestige, mais coût et délai ;
- chapelle privée → plus d’intimité spirituelle, mais conflit paroissial ;
- décor dynastique → plus de légitimité, mais rivalités familiales ;
- trésor itinérant → bonus en voyage, mais risque de perte.

C’est ce qui empêchera la chapelle de devenir une simple succession d’achats donnant des modificateurs permanents. Elle doit être un **organisme coûteux, prestigieux, politiquement utile et continuellement vulnérable**.

- un ou deux chapelains ;
- matériel limité ;
- mobilité élevée.

- premier chapelain ;
- quelques chantres ;

- personnel spécialisé ;

# 17. La boucle stratégique centrale

La véritable boucle ne serait donc plus seulement :

**construire → décorer → améliorer.**

Elle deviendrait :

> **Constituer sa chapelle → répartir son personnel → choisir ce que l’on emporte → voyager → négocier un lieu → déployer la chapelle → utiliser le cérémonial → revenir et réorganiser l’institution.**

Et le dilemme central serait :

> **Plus la chapelle suit efficacement le souverain, moins elle peut remplir ses obligations dans la résidence principale.**

C’est probablement la mécanique la plus importante pour éviter que le déplacement soit seulement cosmétique.
